# Capstone_Project_JantungIn

Brach buat Backend, pake Hapi sama Axios.

## Project Setup

```sh
cd JantungIn-API
```

Atau

```sh
cd JantungIn-API && npm install
```

### Run Debugging

```sh
npm run dev
```

atau check lagi package.json

### Check error with Eslint

```sh
npm run lint
```

Check result, kalau ada yg Error, run

```sh
npx eslit .
```
